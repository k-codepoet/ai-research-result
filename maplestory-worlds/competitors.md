# 메이플스토리 월드 경쟁사 비교 분석

## 1. 경쟁 구도 개요

```
┌─────────────────────────────────────────────────────────────────┐
│                    UGC 게이밍 플랫폼 경쟁 지도                  │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│   Direct Competitors (직접 경쟁)                                │
│   ├── 로블록스 (Roblox) - 시장 리더                            │
│   ├── 포트나이트 크리에이티브 (Epic) - 강력한 2위              │
│   └── 코어 (Manticore) - 후발 주자                             │
│                                                                 │
│   Indirect Competitors (간접 경쟁)                              │
│   ├── 마인크래프트 - 샌드박스 게임                             │
│   ├── 게임 엔진 (Unity, Godot) - 인디 개발                     │
│   └── 소셜 메타버스 (제페토, VRChat)                           │
│                                                                 │
│   Adjacent (인접 시장)                                          │
│   ├── Steam/itch.io - 인디 게임 배포                           │
│   ├── 게임 모딩 커뮤니티                                       │
│   └── 노코드 게임 도구 (GDevelop, Construct)                   │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

## 2. 핵심 경쟁사 상세 분석

### 2.1 로블록스 (Roblox)

#### 기업 개요
| 항목 | 내용 |
|------|------|
| 설립 | 2004년[1] |
| 상장 | NYSE: RBLX (2021 DPO)[1] |
| 시가총액 | $52.43B (2026년 1월)[1] |
| 직원 수 | 2,500+[2] |
| 본사 | 미국 샌마테오 |

#### 핵심 지표 (2024-2025)
```
DAU: 88.9M (Q3 2024), 151.5M (Q3 2025)[1]
MAU: 380M (2024)[1]
Bookings: $4.4B+ (2024 연간)[1]
크리에이터 수: 3.5M+[1]
크리에이터 지급: $922.82M (2024년)[1]
활성 경험 수: 6.7M+[1]
유저 연령: 56% 16세 미만[1]
```

#### 비즈니스 모델
```
수익 구조:
├── Robux 판매: 70%
│   ├── 아이템 구매
│   ├── 게임패스
│   └── 프리미엄 경험
├── 프리미엄 구독: 15%
├── 광고: 10%
└── 기타: 5%

크리에이터 수익 배분:
└── 크리에이터: ~30% (DevEx 환전 기준)[1]
└── 플랫폼: ~70%
```

#### 강점
- ✅ 압도적 규모와 네트워크 효과
- ✅ 성숙한 개발 도구 (Roblox Studio)
- ✅ 강력한 크리에이터 생태계
- ✅ Z세대 문화의 일부
- ✅ 교육 시장 침투

#### 약점
- ❌ 성인 시장 확장 어려움
- ❌ 콘텐츠 모더레이션 이슈[2]
- ❌ 크리에이터 수익 배분 논란
- ❌ 고품질 그래픽 한계
- ❌ 아시아 시장 상대적 약세

#### 메이플월드 시사점
```
배울 점:
- 크리에이터 도구 투자의 중요성
- 커뮤니티 중심 성장
- 교육 시장 진출 전략

공략 포인트:
- 성인 타겟 (로블록스 약점)
- 아시아 시장 (상대적 갭)
- 2D/레트로 차별화
```

### 2.2 포트나이트 크리에이티브 / UEFN (Epic Games)

#### 개요
| 항목 | 내용 |
|------|------|
| 모회사 | Epic Games (비상장) |
| 밸류에이션 | $32B (2022년 4월)[3] |
| 포트나이트 플레이어 | 125M+ (2018년 6월까지)[4] |
| UEFN 출시 | 2023년 3월[5] |
| 크리에이터 지급 (첫 해) | $320M+ (20,000+ 개발자)[5] |

#### 특징
```
UEFN (Unreal Editor for Fortnite):
├── 언리얼 엔진 5 기반[5]
├── 전문가급 도구
├── 고품질 그래픽
├── 포트나이트 생태계 통합
└── Verse 스크립팅 언어[5]
```

#### 비즈니스 모델
```
크리에이터 수익:
├── 참여 기반 보상 (플레이 시간 비례)
├── 아이템 판매 수익 분배
└── Island Creator 프로그램

특징: 크리에이터에게 40% 매출 배분[5]
```

#### 강점
- ✅ 언리얼 엔진 기술력
- ✅ 업계 최고 그래픽 품질
- ✅ 대형 IP 콜라보 (마블, 스타워즈 등)
- ✅ 관대한 수익 배분 (40%)
- ✅ 10대-20대 코어 유저

#### 약점
- ❌ 배틀로얄 이미지 강함
- ❌ 높은 진입 장벽 (UEFN 난이도)
- ❌ 독립 플랫폼 아님
- ❌ 어린 연령대 타겟 어려움

#### 메이플월드 시사점
```
배울 점:
- 크리에이터 우호적 수익 배분
- IP 콜라보 전략
- 고품질 도구 중요성

공략 포인트:
- 낮은 진입 장벽 (포트나이트 약점)
- 캐주얼 크리에이터
- 독립적 2D 플랫폼 정체성
```

### 2.3 코어 (Core by Manticore Games)

#### 개요
| 항목 | 내용 |
|------|------|
| 설립 | 2016년[6] |
| 펀딩 | $145M+ (Series A-C)[6] |
| 출시 | 2021년 4월 (Early Access)[6] |
| 플랫폼 | PC, Epic Games Store |
| 크리에이터 수익 배분 | 50%[6] |

#### 현황
```
펀딩 내역:[6]
├── 2019년 9월: $30M (Series B)
├── 2020년 9월: $15M (Epic Games 주도)
└── 2021년 3월: $100M (Series C)

상태: 2025년 9월 "Out of Time" 스탠드얼론 게임 출시[6]
```

#### 강점
- ✅ 성인 타겟[6]
- ✅ 언리얼 기반 고품질
- ✅ 50% 크리에이터 수익 배분[6]

#### 약점
- ❌ 유저 기반 작음 [출처 필요]
- ❌ 브랜드 인지도 부족
- ❌ 차별화 부족

#### 메이플월드 시사점
```
교훈:
- "더 나은 로블록스"만으로는 부족
- 명확한 차별화 필수
- IP 없이 후발 진입 어려움

→ 메이플월드는 IP가 핵심 차별점
```

## 4. 경쟁 전략 권고

### 4.1 직접 경쟁 회피

```
❌ 피해야 할 전략:
├── "더 나은 로블록스" 포지셔닝
├── 3D 그래픽 경쟁
├── 글로벌 동시 확장
└── 어린이 시장 정면 대결

✅ 채택할 전략:
├── 2D/픽셀 아트 특화
├── 성인 노스탤지어 타겟
├── 아시아 우선 확장
└── IP 레버리지 극대화
```
