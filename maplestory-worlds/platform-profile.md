# 메이플스토리 월드 플랫폼 개요

## 기본 정보

| 항목 | 내용 |
|------|------|
| 플랫폼명 | 메이플스토리 월드 (MapleStory Worlds) |
| 모회사 | 넥슨 (Nexon) |
| 출시 | 2022년 (베타), 2023년 정식 |
| 플랫폼 | PC (웹 기반), 모바일 지원 |
| 개발 도구 | 메이플스토리 월드 스튜디오 |
| 타겟 | 메이플 팬, 캐주얼 크리에이터 |

## 플랫폼 컨셉

### 핵심 가치 제안
> "누구나 메이플스토리 세계관에서 자신만의 게임을 만들고 공유할 수 있다"

```
┌─────────────────────────────────────────────────────┐
│              메이플스토리 월드 생태계               │
├─────────────────────────────────────────────────────┤
│                                                     │
│   크리에이터 ──► 게임 제작 ──► 퍼블리시 ──► 수익화  │
│        │              │              │              │
│        ▼              ▼              ▼              │
│   스튜디오 도구   메이플 자산    플레이어 커뮤니티  │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 차별점
1. **메이플스토리 IP 활용**: 20년간 축적된 캐릭터, 몬스터, 맵, 사운드
2. **2D 픽셀 아트 특화**: 로블록스(3D)와 차별화
3. **낮은 진입 장벽**: 코딩 없이 드래그앤드롭 방식
4. **노스탤지어 팬덤**: 메이플 등록 유저 1.8억 명+의 잠재 고객[1]

## 플랫폼 구성 요소

### 1. 메이플스토리 월드 스튜디오
```
기능:
├── 맵 에디터
│   ├── 타일 기반 맵 제작
│   ├── 메이플 배경 자산 라이브러리
│   └── 커스텀 에셋 업로드
├── 스크립트 시스템
│   ├── 비주얼 스크립팅 (노드 기반)
│   ├── 루아(Lua) 기반 코딩 옵션
│   └── 이벤트 트리거 시스템
├── 캐릭터/몬스터 에디터
│   ├── 기존 메이플 자산
│   └── 커스텀 스프라이트
└── 테스트 & 디버깅
    ├── 실시간 프리뷰
    └── 멀티플레이 테스트
```

### 2. 마켓플레이스
```
거래 유형:
├── 에셋 판매 (스프라이트, 사운드, 스크립트)
├── 게임 내 아이템 판매
└── 프리미엄 게임 입장권
```

### 3. 플레이어 허브
```
기능:
├── 게임 디스커버리
├── 소셜 기능 (친구, 길드)
├── 리더보드/업적
└── 이벤트/시즌 콘텐츠
```

## 비즈니스 모델 (가정)

### 수익원 구조 (독립 기업 관점)

| 수익원 | 비중 (추정) | 설명 |
|--------|------------|------|
| 인게임 구매 | 40% | 아이템, 코스튬, 화폐 |
| 크리에이터 수수료 | 25% | 에셋/게임 판매의 30% 수수료 |
| 프리미엄 구독 | 20% | 크리에이터 도구, 광고 제거 |
| 광고 | 10% | 무료 유저 대상 |
| IP 라이선스 | 5% | 외부 콜라보 |

### 크리에이터 수익 배분
```
크리에이터 수익 = 총 판매액 × 70%
플랫폼 수수료 = 총 판매액 × 30%

(로블록스 대비 유리: 로블록스는 DevEx 환전 기준 약 30% 크리에이터 몫)[2]
```

## SWOT 분석

### Strengths (강점)
- 메이플스토리 IP 자산 (캐릭터, 세계관, 사운드)
- 20년간의 팬덤과 노스탤지어
- 2D 픽셀 아트 특화 (차별점)
- 넥슨의 운영 노하우와 자금력

### Weaknesses (약점)
- 후발 주자 (로블록스, 포트나이트 대비)
- 글로벌 인지도 부족
- 개발 도구 성숙도 열세
- 크리에이터 생태계 규모 미약

### Opportunities (기회)
- 아시아 시장 성장
- 레트로/픽셀 아트 트렌드
- AI 기반 크리에이션 도구
- 메이플 IP 글로벌 리바이벌

### Threats (위협)
- 로블록스/포트나이트 시장 지배
- 크리에이터 유치 경쟁
- 부적절한 콘텐츠 관리 리스크
- 메이플 본게임과의 카니발라이제이션
