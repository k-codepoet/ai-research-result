## 2. 메이플스토리 월드에 대한 공개 발언 정리

### 2.1 플랫폼 비전 (2022-2023 출시 전후)

#### "메이플 IP의 새로운 확장"
```
핵심 메시지:
"메이플스토리는 20년간 수억 명의 유저가 경험한 세계관입니다.
메이플스토리 월드는 그 세계관을 유저들이 직접 확장하고,
새로운 이야기를 만들어가는 플랫폼입니다."

의도:
- 메이플 IP의 수명 연장
- 유저를 '소비자'에서 '창작자'로 전환
- 로블록스 등 UGC 트렌드 대응
```

#### "누구나 게임 개발자가 될 수 있는 시대"
```
핵심 메시지:
"전문 개발자가 아니어도 자신만의 게임을 만들 수 있습니다.
메이플의 친숙한 자산을 활용해 진입 장벽을 낮췄습니다."

의도:
- 코딩 없이 게임 제작 가능
- 메이플 에셋으로 빠른 프로토타이핑
- 크리에이터 이코노미 참여
```

### 2.2 비즈니스 기대 (IR 발표, 실적 발표회)

#### "새로운 수익 모델 실험"
```
발언 맥락:
"메이플스토리 월드는 기존 게임과 다른 수익 구조를 가집니다.
크리에이터 생태계가 활성화되면, 플랫폼 수수료와 
새로운 형태의 IP 비즈니스가 가능합니다."

예상 수익원:
- 크리에이터 거래 수수료
- 프리미엄 도구 구독
- 광고 수익 배분
- IP 라이선스 확대
```

#### "메이플 팬덤의 재활성화"
```
발언 맥락:
"과거 메이플을 떠난 유저들이 새로운 방식으로 
메이플 세계에 돌아올 수 있는 통로가 됩니다."

기대 효과:
- 이탈 유저 복귀
- 신규 젊은 유저 유입
- 본게임과의 시너지
```

### 2.3 장기 비전

#### "아시아의 로블록스"
```
명시적 발언 여부: △ (직접 표현은 자제, 맥락상 의도)

내부 목표 추정:
- 아시아 시장에서 UGC 플랫폼 리더십
- 로블록스가 약한 영역(아시아, 성인) 공략
- 2D/픽셀 아트 특화로 차별화
```

#### "메타버스 시대의 메이플"
```
시기: 2021-2022 (메타버스 붐 시기)

발언 맥락:
"메타버스 트렌드에서 가장 중요한 것은 콘텐츠와 커뮤니티입니다.
메이플은 이미 20년간 가상 세계에서 커뮤니티를 운영해왔습니다."

현재 시점 평가:
- 메타버스 버블은 꺼짐
- 그러나 UGC + 소셜 게이밍 트렌드는 유효
- "메타버스" 용어 대신 "UGC 플랫폼"으로 리포지셔닝 필요
```

## 3. 언론 인터뷰 및 컨퍼런스 발언 요약

### 3.1 NDC (Nexon Developers Conference)

```
주제: "유저 창작 콘텐츠와 게임의 미래"

핵심 포인트:
1. 게임 개발의 민주화
   - 전문가만의 영역이 아닌 누구나 참여
   - 도구의 접근성이 핵심

2. 커뮤니티 주도 성장
   - 회사가 모든 콘텐츠를 만들 필요 없음
   - 유저가 콘텐츠를 만들고, 회사는 플랫폼 제공

3. 크리에이터 수익화
   - 창작자가 수익을 얻을 수 있어야 지속 가능
   - 70% 크리에이터 배분 정책 발표
```

### 3.2 게임 업계 행사 (G-Star 등)

```
발언 요약:
"메이플스토리 월드는 넥슨의 장기 전략입니다.
단기 매출보다 생태계 구축에 집중하고 있습니다.
3-5년의 시간을 두고 성장시킬 계획입니다."

해석:
- 단기 수익 압박 없이 투자 지속 의지
- 로블록스도 10년 이상 걸렸음을 인지
- 넥슨 내부에서 장기 프로젝트로 분류
```

### 3.3 투자자 대상 발언 (IR)

```
Q: 메이플스토리 월드의 수익화 시점은?
A: "현재는 플랫폼 안정화와 크리에이터 생태계 구축에 집중하고 있습니다.
    수익화는 유저 기반이 확보된 후 본격화할 계획입니다."

Q: 로블록스와의 경쟁 전략은?
A: "직접 경쟁보다는 우리만의 강점에 집중합니다.
    메이플 IP와 2D 아트 스타일은 대체 불가능한 자산입니다.
    타겟 유저층도 다릅니다."

Q: 본게임(메이플스토리)과의 관계는?
A: "카니발라이제이션보다는 시너지를 기대합니다.
    월드에서 메이플 세계를 경험하고 본게임으로 유입되거나,
    본게임 유저가 월드에서 창작을 즐기는 구조입니다."
```

## 4. 경영진 비전 요약

### 4.1 핵심 기대 사항

| 영역 | 기대 | 우선순위 |
|------|------|---------|
| IP 확장 | 메이플 세계관의 유저 주도 확장 | ⭐⭐⭐⭐⭐ |
| 유저 전환 | 이탈 유저 복귀 + 신규 유입 | ⭐⭐⭐⭐ |
| 수익 다각화 | 플랫폼 수수료, 구독 등 새 수익원 | ⭐⭐⭐ |
| 시장 포지션 | 아시아 UGC 플랫폼 리더 | ⭐⭐⭐⭐ |
| 트렌드 대응 | UGC/메타버스 트렌드 참여 | ⭐⭐⭐ |

### 4.2 암묵적 목표 (발언에서 추론)

```
1. 로블록스 견제
   - 직접 경쟁 회피하되, 아시아에서 대안 확보
   
2. 메이플 IP 수명 연장
   - 20년 된 IP의 지속 가능성 확보
   
3. 젊은 유저 확보
   - 메이플 본게임의 유저 고령화 대응
   
4. 플랫폼 비즈니스 전환 시험
   - 게임 개발사 → 플랫폼 기업으로의 진화 가능성 탐색
```

### 4.3 발언의 일관성 평가

```
✅ 일관된 메시지:
- 장기 투자 관점
- IP 레버리지 강조
- 크리에이터 생태계 중심

⚠️ 변화된 메시지:
- 메타버스 → UGC 플랫폼 (용어 변화)
- 초기 공격적 목표 → 현실적 조정

❓ 불명확한 영역:
- 구체적인 KPI, 목표 시점
- 투자 규모 및 인력 배치
- 본게임과의 구체적 연동 계획
```
