## 2. 콘텐츠 생태계 현황

### 2.1 인기 월드 유형 분석

```
월드 장르 분포:

RPG/어드벤처: 35%
├── 메이플 스토리 재현
├── 팬메이드 스토리
└── 간단한 던전

미니게임: 30%
├── 점프맵
├── 피하기 게임
├── 레이싱

소셜/힐링: 20%
├── 타운 시뮬레이션
├── 꾸미기
└── 채팅방

기타: 15%
├── 퍼즐
├── 호러
└── 실험적
```

### 2.2 콘텐츠 품질 분포

```
품질 등급별 월드 수:

S급 (바이럴 가능): ~50개 (0.1%)
A급 (높은 완성도): ~500개 (1%)
B급 (플레이 가능): ~5,000개 (10%)
C급 (미완성/테스트): ~45,000개 (90%)

문제점:
- S급 콘텐츠 절대 부족
- 바이럴 히트작 부재
- 상위 콘텐츠도 로블록스 인기작 대비 품질 격차
```

### 2.3 크리에이터 생태계

```
크리에이터 티어 구조:

Top Creator (상위 0.1%)
├── 인원: ~30-50명
├── 월 수익: $1,000-5,000
├── 특징: 전문성 있으나 전업하기 어려움
└── 문제: 이탈 위험 (다른 플랫폼으로)

Active Creator (상위 5%)
├── 인원: ~1,000-3,000명
├── 월 수익: $50-500
├── 특징: 취미 수준, 지속성 불확실
└── 문제: 동기 부여 필요

Casual Creator (나머지)
├── 인원: ~10,000+
├── 월 수익: 거의 없음
├── 특징: 한두 번 시도 후 이탈
└── 문제: 높은 이탈률
```
