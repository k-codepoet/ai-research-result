# 메이플스토리 월드 전략 제언

## Executive Summary

메이플스토리 월드가 경영진 비전을 달성하기 위해서는 **크리에이터 생태계 구축**에 집중 투자해야 합니다. 현재 가장 큰 병목은 "유저 부족 → 크리에이터 수익 부족 → 크리에이터 이탈 → 콘텐츠 부족 → 유저 이탈"의 악순환입니다.

**핵심 제언: "100명의 스타 크리에이터를 만들어라"**

---

## 1. 전략적 프레임워크

### 1.1 전략 방향

```
┌─────────────────────────────────────────────────────────────┐
│                                                             │
│   현재: 플랫폼 개선 → 유저 유입 → 크리에이터 성장 (실패)   │
│                                                             │
│   전환: 크리에이터 육성 → 히트 콘텐츠 → 유저 유입 → 선순환 │
│                                                             │
└─────────────────────────────────────────────────────────────┘

핵심 전환:
"플랫폼을 만들면 크리에이터가 온다" (X)
"크리에이터를 키우면 플랫폼이 된다" (O)
```

### 1.2 3개년 전략 로드맵

```
Year 1: IGNITE (점화)
├── 목표: 크리에이터 생태계 기반 구축
├── 핵심 KPI: 스타 크리에이터 100명, DAU 500K
├── 투자: 현재 대비 2배
└── 테마: "100명의 스타를 찾아라"

Year 2: ACCELERATE (가속)
├── 목표: 유저 성장 가속화
├── 핵심 KPI: DAU 1.5M, 히트작 10개
├── 투자: Year 1 성과에 따라 결정
└── 테마: "메이플 월드, 모두의 놀이터"

Year 3: EXPAND (확장)
├── 목표: 아시아 리더십 확보
├── 핵심 KPI: 글로벌 DAU 3M
├── 투자: 적극 확대
└── 테마: "아시아의 UGC 허브"
```

## 2. Year 1 핵심 전략: IGNITE

### 2.1 전략 1: 크리에이터 100 프로젝트

**목표**: 1년 내 풀타임 또는 세미 풀타임 크리에이터 100명 육성

```
프로그램 구조:

[크리에이터 100 펀드]
├── 총 규모: $5M (약 65억 원)
├── 지원 방식:
│   ├── 월 고정 지원금: $2,000-5,000/인
│   ├── 성과 보너스: DAU/매출 연동
│   └── 제작 지원: 에셋, 기술 지원
├── 선발: 공개 모집 + 스카우트
└── 기간: 12개월 계약

[선발 기준]
├── 기존 크리에이터 (메이플 월드, 로블록스, 유니티 등)
├── 유튜버/스트리머 (게임 콘텐츠)
├── 인디 게임 개발자
└── 메이플 커뮤니티 인플루언서

[지원 프로그램]
├── 전담 매니저 배정 (1:5-10)
├── 기술 멘토링
├── 마케팅 지원 (공식 채널 노출)
├── 오프라인 스튜디오 제공
└── 네트워킹 이벤트
```

**예상 효과**:
- 100명 × 평균 5개 월드 = 500개 고품질 월드
- 이 중 10%가 히트 → 50개 인기 월드
- 크리에이터의 팬덤 유입 → 신규 유저

### 2.2 전략 2: 히트 콘텐츠 부스팅

**목표**: 바이럴 가능한 히트 콘텐츠 10개 확보

```
방법 1: 자체 제작 (Nexon Studio)
├── 투입: 전담 팀 10-20명
├── 제작: 공식 미니게임 시리즈
├── 품질: 상용 게임 수준
├── 목적: 플랫폼 쇼케이스
└── 예산: $2M

방법 2: 크리에이터 후원
├── 상위 콘텐츠 선별
├── 추가 개발 지원
├── 마케팅 지원
└── 예산: $1M

방법 3: IP 콜라보
├── 한국 웹툰/애니 IP
├── 넥슨 다른 게임 IP
├── 외부 브랜드
└── 예산: $1M

방법 4: 콘테스트
├── 분기별 대규모 크리에이터 대회
├── 상금 총 $500K/년
├── 미디어 노출
└── 예산: $500K
```

### 2.3 전략 3: 인지도 캠페인

**목표**: 한국 게이머 인지도 10% → 30% [AI 추정, 2024년 학습 데이터 기반]

```
캠페인: "메이플에서 놀자, 내가 만든 세상"

[페이즈 1: 티저] (1-2개월)

[페이즈 3: 지속] (5-12개월)

├── 크리에이터 스포트라이트
├── 커뮤니티 빌딩
└── UGC 마케팅 (유저 콘텐츠 활용)

[예산 배분]
├── 디지털 광고: $2M
├── 인플루언서: $1M
├── PR/이벤트: $500K
├── 콘텐츠 제작: $500K
└── 총: $4M
```

### 2.4 전략 4: AI 크리에이션 도구 (MVP)

**목표**: 진입 장벽을 획기적으로 낮추는 AI 도구

```
[MVP 기능]
├── 텍스트 → 맵 생성
│   └── "벚꽃이 흩날리는 마을을 만들어줘"
├── AI NPC 대화
│   └── 기본 대화 자동 생성
└── 자동 밸런싱 제안
    └── 게임 난이도 조절 힌트

[개발 계획]
├── 기간: 6-9개월
├── 팀: 10-15명 (ML 엔지니어 포함)
├── 예산: $2M
└── 기술: LLM + 게임 에셋 매핑

[기대 효과]
├── 첫 월드 제작 시간: 2시간 → 30분
├── 크리에이터 전환율: 5% → 15%
└── "코딩 없이 게임 만들기" 실현
```

### 2.5 전략 5: 본게임 시너지 강화

**목표**: 본게임 ↔ 월드 양방향 시너지

```
[양방향 보상 시스템]
├── 월드 → 본게임
│   ├── 월드 플레이 → 본게임 재화/아이템
│   ├── 월드 업적 → 본게임 칭호
│   └── 월드 크리에이터 → 본게임 특별 보상
│
└── 본게임 → 월드
    ├── 본게임 레벨/업적 → 월드 특전
    ├── 본게임 캐릭터 → 월드 아바타
    └── 본게임 이벤트 → 월드 연동

[통합 이벤트]
├── 분기 1회 대규모 크로스 이벤트
├── 스토리 연동 (세계관 확장)
└── 공동 마케팅

[조직 연계]
├── 본게임-월드 합동 기획 회의 (월 1회)
├── KPI 공유 (시너지 지표)
└── 공동 목표 설정
```
