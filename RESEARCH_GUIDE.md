# 기업 분석 리서치 가이드

> 이 문서는 시장/기업 분석 문서 작성 시 반드시 따라야 할 절차를 정의합니다.

## 핵심 원칙

```
⚠️ 출처 없는 데이터는 신뢰할 수 없다.
```

모든 수치, 통계, 팩트는 **검증 가능한 출처**와 함께 기록해야 합니다.

---

## 작업 절차

### Phase 1: 초안 작성 (Draft)

1. AI 학습 데이터 기반으로 문서 구조와 초안 작성
2. 모든 데이터 포인트에 `[미검증]` 태그 표시
3. 검증이 필요한 항목 목록 생성

```markdown
예시:
- 시장 규모: $50B [미검증]
- 성장률: 15% CAGR [미검증]
```

### Phase 2: 출처 검증 (Verification)

1. 각 `[미검증]` 항목에 대해 웹 검색 수행
2. 신뢰할 수 있는 출처에서 데이터 확인
3. 출처 번호 `[1]`, `[2]` 형식으로 본문에 표시
4. 문서 하단에 `## 출처` 섹션 추가

```markdown
예시:
- 시장 규모: $50B[1]

## 출처
[1] Grand View Research, "Market Report 2024", https://example.com/report
```

### Phase 3: 검증 불가 처리

출처를 찾을 수 없는 경우:

| 상황 | 처리 방법 |
|------|----------|
| 공식 데이터 없음 | `[AI 추정, 2024년 학습 데이터 기반]` 표시 |
| 유료 리포트만 있음 | 리포트명 + "유료" 표시 |
| 내부 자료 필요 | `[내부 자료 필요]` 표시 후 담당자 확인 요청 |

---

## 신뢰할 수 있는 출처 우선순위

### Tier 1 (최우선)
- 기업 공식 IR 자료, 연간 보고서
- 정부 통계 (KOSIS, e-나라지표)
- SEC/DART 공시 자료

### Tier 2 (권장)
- Wikipedia (인용 출처 확인 필수)
- Bloomberg, Reuters
- 유명 리서치 기관 (McKinsey, Gartner, Grand View Research)

### Tier 3 (참고용)
- 언론 기사 (날짜 명시)
- 업계 블로그/분석
- LinkedIn, 기업 블로그

### 비권장
- 출처 불명 블로그
- SNS 게시물
- AI 생성 콘텐츠 (자기 자신 포함)

## 출처 표기 형식

### 본문 내 표기
```markdown
글로벌 게임 시장 규모는 약 $299B[1]이며, 
모바일이 49%[2]를 차지한다.
```

### 하단 출처 섹션
```markdown
## 출처

[1] Grand View Research, "Video Game Market Size Report 2024"
    https://www.grandviewresearch.com/industry-analysis/video-game-market

[2] Mordor Intelligence, "Gaming Market Analysis 2025"
    https://www.mordorintelligence.com/industry-reports/global-gaming-market

[3] AI 추정, 2024년 학습 데이터 기반 (검증 필요)
```

## 체크리스트

문서 완료 전 확인:

- [ ] 모든 수치에 출처 번호가 있는가?
- [ ] 출처 섹션에 모든 번호가 정의되어 있는가?
- [ ] 검증 불가 항목이 명확히 표시되어 있는가?
- [ ] 출처 URL이 접근 가능한가?
- [ ] 출처 날짜가 최신인가? (2년 이내 권장)

## 작업 명령 예시

### Claude/OpenClaw에게 지시할 때

**초안 작성:**
```
[기업명] 시장 분석 초안을 작성해줘.
모든 데이터 포인트에 [미검증] 태그를 달아줘.
```

**검증 작업:**
```
작성된 문서의 [미검증] 항목들을 웹 검색으로 검증해줘.
출처를 찾으면 각주 형식으로 추가하고,
못 찾으면 [AI 추정, 2024년 학습 데이터 기반]으로 표시해.
```
