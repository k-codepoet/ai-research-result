## 3. 경쟁 환경

### 3.1 직접 경쟁사 분석

#### 텐센트 (Tencent)
```
강점:
- 압도적 규모 (2024 매출 $91.8B)[4]
- 위챗 생태계
- 세계 최대 게임 기업[4]
- Riot Games, Epic Games 지분 보유[4]

약점:
- 중국 규제 리스크 (미성년자 게임 시간 제한 등)[3]
- 서구 시장 직접 운영 한계

넥슨 대비:
- 규모에서 크게 열세
- 하지만 특정 장르(횡스크롤 RPG)에서 경쟁력
```

#### EA (Electronic Arts)
```
강점:
- 스포츠 게임 독점 (FIFA, NFL)
- 서구 시장 강자
- 라이브 서비스 전환 중

약점:
- 아시아 약세
- 모바일 부진

넥슨 대비:
- FIFA 온라인은 아시아 특화로 공존
- 다른 영역에서 직접 경쟁 적음
```

#### 엔씨소프트
```
강점:
- 리니지 IP 파워
- MMORPG 전문성
- AI 연구 (NC AI)

약점:
- 리니지 의존도
- 신작 부진
- 해외 확장 어려움

넥슨 대비:
- 장르 다양성에서 넥슨 우위
- AI에서는 비슷한 수준
```

## 5. SWOT 분석

### Strengths (강점)
- 20년+ 장수 IP 포트폴리오
- 라이브 서비스 운영 노하우
- 방대한 유저 데이터 (던파 8.5억명+)[6]
- 현금 여력 풍부 [출처 필요]
- 중국 시장 입지 (던파)

### Weaknesses (약점)
- 신작 히트작 부재
- 글로벌 확장 더딤
- 모바일 시장 경쟁력 약화
- AI 전환 속도 느림
- 조직 사일로

### Opportunities (기회)
- AI 기반 게임 혁신
- B2B 데이터 플랫폼
- M&A 기회 (현금 활용)
- 신흥 시장 (동남아, 중동)[3]
- 콘솔 시장 진출

### Threats (위협)
- 텐센트/호요버스 경쟁 심화
- 주력 IP 노화
- 중국 시장 리스크[3]
- 인재 유출
- 규제 강화

## 6. 투자 관점 분석

### 6.1 Valuation [출처 필요]

| 지표 | 넥슨 | 업계 평균 |
|------|------|----------|
| P/E | 15-20x | 18-25x |
| EV/EBITDA | 8-12x | 10-15x |
| P/S | 3-5x | 4-6x |

**평가**: 현재 밸류에이션은 성장 둔화 반영, AI 전환 성공 시 리레이팅 가능

**참고**: 정확한 밸류에이션 지표는 Yahoo Finance, Bloomberg 등 금융 데이터 제공자 참조 필요.

### 6.2 투자 포인트

**Bull Case**:
- AI 전환으로 효율성 대폭 개선
- 신작 히트 (던파 모바일 글로벌 등)
- B2B 플랫폼 사업 진출
- 목표: 현재 대비 +50-100%

**Bear Case**:
- 주력 IP 매출 감소 지속
- AI 전환 지연
- 중국 리스크 현실화
- 하방: 현재 대비 -20-30%
